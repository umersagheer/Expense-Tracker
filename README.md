## Run Locally

Clone the project

```bash
  git clone https://github.com/umersagheer75/Expense-Tracker.git
```

Go to the project directory

Install dependencies[in ROOT and in CLIENT]

```bash
  npm install
```

Start the server[in ROOT dir]

```bash
  npm run server
```

Start the client[in client dir]

```bash
  npm run dev
```
